# ECO2 🌿

> *Cuidamos el planeta, una planta a la vez.*

**ECO2** es una aplicación móvil multiplataforma que integra Inteligencia Artificial para identificar especies de plantas domésticas a partir de fotografías, brindando cuidados personalizados y promoviendo hábitos de vida sostenibles mediante gamificación ecológica.

---

## 🌱 Sobre el Proyecto

ECO2 nace como respuesta a una necesidad real: la mayoría de los aficionados a las plantas domésticas no cuenta con los conocimientos botánicos necesarios para identificar las especies que poseen ni para aplicar los cuidados adecuados. Esto deriva en la pérdida frecuente de plantas por negligencia involuntaria.

El nombre **ECO2** combina el prefijo *ECO* (ecología, ecosistema, economía sostenible) con el número *2*, en referencia a la fórmula química del dióxido de carbono (CO₂), cuya reducción es uno de los propósitos ambientales centrales de la aplicación.

### Objetivo General

Desarrollar una aplicación móvil multiplataforma (Android e iOS) que integre un modelo de IA de visión por computadora para identificar especies de plantas domésticas a partir de fotografías, brindando información detallada sobre sus cuidados personalizados y promoviendo hábitos sostenibles.

---

## ✨ Funcionalidades

- 📸 **Escáner IA de plantas** — Identificación de 50+ especies domésticas con precisión superior al 85%
- 📖 **Catálogo botánico** — Información de cuidados: riego, luz, temperatura, sustrato y abonado
- 🎮 **Gamificación ecológica** — Misiones diarias, trofeos, niveles y moneda virtual (*semillas*)
- 🌍 **Calculadora de CO₂** — Estimación de absorción de carbono según las plantas del hogar
- 🔔 **Notificaciones inteligentes** — Recordatorios personalizados de cuidados pendientes
- 🔬 **Catálogo enciclopédico** — Origen geográfico, familia botánica y nombre científico por especie

## 🛠️ Tecnologías

| Tecnología | Capa | Descripción |
|---|---|---|
| **Flutter (Dart)** | Frontend / Mobile | Framework multiplataforma para Android e iOS con rendimiento nativo |
| **Express + Node.js** | Backend / API REST | Servidor web minimalista y flexible para gestión de rutas HTTP |
| **TensorFlow / Keras** | Inteligencia Artificial | Entrenamiento y despliegue del modelo de clasificación de imágenes |
| **PostgreSQL / Firebase** | Base de Datos / Auth | Soporte para búsquedas semánticas con pgvector e integración con IA |
| **Google ML Kit** | Visión on-device | Procesamiento de imágenes en dispositivo sin latencia de red |
| **Azure** | Infraestructura Cloud | Alojamiento del backend y base de datos con CI/CD via GitHub |
| **Figma** | Diseño UX/UI | Prototipado colaborativo, wireframes y sistema de diseño |
| **GitHub** | Control de Versiones | Repositorio centralizado con gestión de ramas y revisión de código |
